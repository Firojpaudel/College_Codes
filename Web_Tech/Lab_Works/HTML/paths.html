Path Types Explained

  • Absolute Path: https://www.example.com/images/photo.jpg
  • Relative Path (same directory): image.jpg
  • Relative Path (subdirectory): images/photo.jpg
  • Relative Path (parent directory): ../images/photo.jpg
  • Root Relative Path: /images/photo.jpg
