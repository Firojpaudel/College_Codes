## 🚀 How to Run

1. Clone this repository:
   ```bash
   git clone <repository-url>
   cd web-technology-lab
   ```

2. For PHP files:
   - Ensure PHP (7+) and MySQL are installed
   - Configure database in `PHP/includes/config.php`
   - Run database setup:
     ```bash
     php setup_database.php
     ```
   - Default credentials:
     ```
     Username: admin
     Password: admin123
     ```
