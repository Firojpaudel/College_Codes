HTML Introduction

1. About

HTML stands for Hyper Text Markup Language. It is used to design web pages. It is a standard markup language for creating web pages.

2. All HTML tags available:

  1. <!DOCTYPE> - Defines the document type
  2. <html> - Root element of an HTML page
  3. <head> - Contains meta-information about the document
  4. <title> - Sets the title of the document
  5. <meta> - Defines metadata about an HTML document
  6. <link> - Defines the relationship between a document and an external resource
  7. <style> - Contains style information for a document
  8. <script> - Defines client-side JavaScript
  9. <body> - Contains the content of an HTML document
  10. <header> - Defines a header for a document or section
  11. <nav> - Defines navigation links
  12. <main> - Specifies the main content of a document
  13. <section> - Defines a section in a document
  14. <article> - Defines an independent, self-contained content
  15. <aside> - Defines content aside from the content it is placed in
  16. <footer> - Defines a footer for a document or section
  17. <h1> to <h6> - Defines HTML headings
  18. <p> - Defines a paragraph
  19. <br> - Inserts a single line break
  20. <hr> - Defines a thematic change in the content
  21. <!--...--> - Defines a comment
  22. <a> - Defines a hyperlink
  23. <img> - Embeds an image
  24. <div> - Defines a division or section
  25. <span> - Defines a section in a document
  26. <ul> - Defines an unordered list
  27. <ol> - Defines an ordered list
  28. <li> - Defines a list item
  29. <table> - Defines a table
  30. <tr> - Defines a row in a table
  31. <td> - Defines a cell in a table
  32. <th> - Defines a header cell in a table
  33. <form> - Defines an HTML form for user input
  34. <input> - Defines an input control
  35. <button> - Defines a clickable button
  36. <select> - Defines a drop-down list
  37. <option> - Defines an option in a drop-down list
  38. <textarea> - Defines a multi-line text input control
  39. <label> - Defines a label for an input element
  40. <fieldset> - Groups related elements in a form
  41. <legend> - Defines a caption for a <fieldset> element
  42. <iframe> - Defines an inline frame
  43. <canvas> - Used to draw graphics via scripting
  44. <svg> - Defines vector-based graphics
  45. <audio> - Embeds sound content
  46. <video> - Embeds video content
  47. <source> - Specifies multiple media resources
  48. <track> - Defines text tracks for media elements
  49. <embed> - Embeds external content
  50. <object> - Defines an embedded object
  51. <param> - Defines parameters for an object
  52. <map> - Defines an image map
  53. <area> - Defines an area inside an image map
  54. <blockquote> - Defines a section that is quoted from another source
  55. <q> - Defines a short quotation
  56. <cite> - Defines the title of a work
  57. <code> - Defines a piece of computer code
  58. <pre> - Defines preformatted text
  59. <em> - Defines emphasized text
