# Advanced Java Programming Lab Solutions

This folder contains the practical task solutions for the Advanced Java topics. 

The structure is classified by Questions:
- `Q1_SwingStudentReg` -> Swing student registration
- `Q2_TCP` -> Simple TCP communication
- `Q3_UDP` -> UDP basic messaging 
- `Q4_FileTransfer` -> File over TCP streams
- `Q5_URLAnalyzer` -> Using `java.net.URL`
- `Q6_EmailSend` -> JavaMail sender
- `Q7_EmailRead` -> JavaMail IMAP reader
- `Q8_JavaFXLibrary` -> JavaFX minimal UI
- `Q9_BasicServlet` -> Understanding servlet lifecycle
- `Q10_ServletGET` -> Basic GET manipulation
- `Q11_ServletRegistration` -> HTML + Servlet
- `Q12_ServletLogin` -> Simple servlet login
- `Q13_EmployeeSession` -> Session storage
- `Q14_JSPRegistration` -> Raw JSP form + JDBC connection
- `Q15_OnlineExam` -> Small JSP assessment 
- `Q16_RMI` -> RMI server/client implementation

_Note: For the web components (Q9-15), you'll need an application server like Apache Tomcat, and the JavaMail tasks (Q6, Q7) require `javax.mail` dependencies on your build path._
